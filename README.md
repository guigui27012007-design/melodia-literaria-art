# 🌸 Melodia Literária

> Onde as histórias se transformam em abraços

Website oficial da **Melodia Literária** — um refúgio para leitores criado por **Yasmin Vitória**.

## 💜 Sobre

A Melodia Literária é o sonho de uma garota de 19 anos que foi salva pela leitura desde criança. Esse sonho toma forma como um refúgio para todos que amam perder-se nas páginas de um bom livro.

## 🎯 Missão

Acolher pessoas que já se sentiram sozinhas demais no mundo. Criar um espaço onde os livros abraçam, cada história aproxima o leitor e todos encontram companhia, cura e pertencimento.

## 💎 Valores

- Acolhimento
- Sensibilidade
- Conexão
- Liberdade Criativa
- Refúgio Seguro
- Transformação
- Inclusão
